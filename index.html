About Martynak Layout

Martynak is a set of optimized keyboard layouts developed in 2018 to solve the problem of efficient touch typing when switching between Russian and English languages.

The layout is based on the fully optimized English QGMLWY (Carpalx) layout and adapted for comfortable work with Cyrillic through phonetic correspondences.

Over 8 years of development, the layout has gone through many iterations of improvements, becoming an efficient tool for programmers and writers.

Metrics based on Carpalx research for QGMLWY English layout

Advantages over QWERTY:

  • 44% reduction in overall effort
  • Decreased pinky load from 16% to 11%
  • Increased home row usage to 74%
  • Improved hand alternation

Comparison with Colemak:

  • 10% less overall effort
  • Better hand alternation (60% vs 55%)
  • Lower pinky strain
  • Preserves ZXCV for shortcuts

üöÄ Getting Started

Step-by-step instructions for learning the Martynak layout

üìö Learning on macOS

We recommend using the KeyKey app to learn the English part of the layout. The Russian layout has high mnemonic compatibility, so you don't need to learn it separately.

Steps:

  1. Install KeyKey from the App Store
  2. Select the QGMLWY layout
  3. Practice 15-30 minutes daily
  4. Switch to Russian to reinforce learning
