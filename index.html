О раскладке Мартынак

Мартынак — это набор оптимизированных клавиатурных раскладок, разработанный в 2018 году для решения проблемы эффективной слепой печати при переключении между русским и английским языками.

Раскладка основана на полностью оптимизированной английской раскладке QGMLWY (Carpalx) и адаптирована для комфортной работы с кириллицей через фонетические соответствия.

За 8 лет развития раскладка прошла множество итераций улучшений, став эффективным инструментом для программистов и писателей.

Статистика эффективности

Метрики основаны на исследованиях Carpalx для английской раскладки QGMLWY

44% снижение усилий vs QWERTY
74% печати на домашнем ряду
11% использование мизинца
6% печати на нижнем ряду

Преимущества перед QWERTY:

  • Снижение общих усилий на 44%
  • Уменьшение нагрузки на мизинец с 16% до 11%
  • Увеличение использования домашнего ряда до 74%
  • Улучшенное чередование рук

Сравнение с Colemak:

  • На 10% меньше общих усилий
  • Лучшее чередование рук (60% vs 55%)
  • Меньшая нагрузка на мизинец
  • Сохранение ZXCV для горячих клавиш

About Martynak Layout

Martynak is a set of optimized keyboard layouts developed in 2018 to solve the problem of efficient touch typing when switching between Russian and English languages.

The layout is based on the fully optimized English QGMLWY (Carpalx) layout and adapted for comfortable work with Cyrillic through phonetic correspondences.

Over 8 years of development, the layout has gone through many iterations of improvements, becoming an efficient tool for programmers and writers.

🚀 Getting Started

Step-by-step instructions for learning the Martynak layout

📚 Learning on macOS

We recommend using the KeyKey app to learn the English part of the layout. The Russian layout has high mnemonic compatibility, so you don't need to learn it separately.

Steps:

  1. Install KeyKey from the App Store
  2. Select the QGMLWY layout
  3. Practice 15-30 minutes daily
  4. Switch to Russian to reinforce learning
